◆ labpack_reader_begin_array_or_nil()

bool labpack_reader_begin_array_or_nil ( labpack_reader_t *  reader,
uint32_t *  count 
)

Begins reading an array or nil.

The elements count is stored in count. The labpack_reader_end_array function should be called after reading all of the elements unless the value was nil instead of an array.

Returns true if an array was read; otherwise, false if nil is return.

The decoder is placed into an error state if the type is not an array or nil. In the error state, the return value is false.

◆ labpack_reader_begin_bin()

uint32_t labpack_reader_begin_bin ( labpack_reader_t *  reader)

Begins reading a binary blob.

The labpack_read_bytes function should be used after this function to read the bytes of the blob and then completed with the labpack_reader_end_bin function.

Returns the bytes count of the binary blob or zero (0) if an error occurred.

The decoder is placed into an error status if the type is not a binary blob.

◆ labpack_reader_begin_ext()

uint32_t labpack_reader_begin_ext ( labpack_reader_t *  reader,
int8_t *  type 
)

Begins reading an extension type.

The labpack_read_bytes function should be used after this function to read the data bytes of the extension type and then completed with the labpack_reader_end_ext function.

Returns the bytes count of the data for the extension or zero (0) if an error occurred. The extension type is passed to the type.

The decoder is placed into an error status if the type is not an extension type.

Reads a map or nil.

The key-value pairs count is stored in count. The labpack_reader_end_map function should be called once all of the key-value pairs have been read unless the value was nil.

Returns true if a map was read; otherwise, false.

The decoder is placed into an error state if the type is not a map or nil. In the error state, the return value is false.
